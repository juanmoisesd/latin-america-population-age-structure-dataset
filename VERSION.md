# Dataset Versioning

## Version 1.2.0 (Current)

Expanded demographic dataset including:

* 42 demographic variables
* population age structure in 15 quinquennial age groups
* sex disaggregation (men, women, total)
* demographic indicators (fertility, life expectancy, migration, urbanization, dependency ratios)
* population counts and percentage distributions by age group

Coverage:

* 19 Latin American countries
* 1995–2030
* quinquennial observations

Total observations:

456 country–year–sex records

## Version 1.1.0

Initial structured demographic dataset including:

* population totals
* broad age structure indicators
* derived demographic indicators

## Version 1.0.0

First compiled dataset structure used during initial data harmonization and validation.
