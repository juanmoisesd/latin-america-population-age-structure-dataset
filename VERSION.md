# Dataset Version History

## Version 1.1.0 (Current — March 2026)

**This is the current stable release.**

Changes in v1.1.0:
- Expanded coverage from initial structured release
- 41 demographic variables per record
- 19 Latin American countries
- Coverage period 1995–2030
- 228 country-year records
- Added derived indicators: dependency ratio, aging index, median age
- Added complete documentation suite (KNOWN_ISSUES.md, DATA_PAPER.md, etc.)
- Unified all metadata to CC0 1.0 license
- Fixed inconsistencies in country count and period references across all files

**Files:** `data/latin_america_population_age_structure.csv`
