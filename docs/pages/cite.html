Cómo citar este dataset

Este proyecto proporciona un dataset demográfico sobre la estructura de edad de la población en América Latina. Para garantizar la correcta atribución académica, se recomienda citar el dataset utilizando alguno de los formatos siguientes.

Cita recomendada (APA)

De la Serna, J. M. (2026). Latin America Population Age Structure Dataset (2000–2023). Zenodo. https://doi.org/10.5281/zenodo.18891177

Cita BibTeX


@dataset{delaserna2026latinamerica,
  author       = {Juan Moisés de la Serna},
  title        = {Latin America Population Age Structure Dataset (2000–2023)},
  year         = {2026},
  publisher    = {Zenodo},
  doi          = {10.5281/zenodo.18891177},
  url          = {https://juanmoisesd.github.io/latin-america-population-age-structure-dataset/}
}
