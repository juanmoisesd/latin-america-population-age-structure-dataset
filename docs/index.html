Qué ofrece este sitio

Páginas específicas por país, fichas anuales, series por indicador, comparaciones entre países, rankings temáticos y páginas formuladas como preguntas de investigación.

Cada sección enlaza con el dataset principal, el proyecto en OSF y la forma correcta de citar el recurso.
