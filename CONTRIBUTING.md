# Contributing to Latin America Demographic Data

We welcome contributions! To contribute:

1. **Bug Reports & Feature Requests**: Open an issue on GitHub.
2. **Data Updates**: If you have more recent official data, please submit a CSV update via Pull Request.
3. **Visualization Improvements**: We use Matplotlib and Seaborn. New plotting scripts should be added to `scripts/`.

## Code of Conduct
Please be respectful and ensure all data sources are properly cited.
