
# Data Dictionary

## Country
Type: string  
Description: Name of the country.

## Year
Type: integer  
Description: Reference year of the demographic observation.

## Population_Total_Millions
Type: numeric  
Description: Total population expressed in millions.

## Pct_0_14
Type: percentage  
Description: Share of population aged 0–14 years.

## Pct_15_24
Type: percentage  
Description: Share of population aged 15–24 years.

## Pct_25_54
Type: percentage  
Description: Share of population aged 25–54 years.

## Pct_55_64
Type: percentage  
Description: Share of population aged 55–64 years.

## Pct_65_plus
Type: percentage  
Description: Share of population aged 65 years or older.

## Pop_0_14
Type: numeric  
Description: Absolute population aged 0–14 (thousands).

## Pop_15_24
Type: numeric  
Description: Absolute population aged 15–24 (thousands).

## Pop_25_54
Type: numeric  
Description: Absolute population aged 25–54 (thousands).

## Pop_55_64
Type: numeric  
Description: Absolute population aged 55–64 (thousands).

## Pop_65_plus
Type: numeric  
Description: Absolute population aged 65+ (thousands).

## Source
Type: string  
Description: Data source used to compile the dataset (e.g., CEPAL, World Bank, BBVA Research).
