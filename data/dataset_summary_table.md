# Resumen del dataset

## Cobertura general

| Característica | Valor |
|---|---|
| Región | América Latina |
| Periodo temporal | 1995–2030 |
| Tipo de datos | Demográficos |
| Unidad de observación | País + Año |
| Formato principal | CSV |

---

## Variables principales

| Variable | Descripción |
|---|---|
| PasAo | País |
| Año | Año de observación |
| PoblaciónTotalMillones | Población total |
| Pct014 | Población 0–14 años |
| Pct1524 | Población 15–24 años |
| Pct2554 | Población 25–54 años |
| Pct5564 | Población 55–64 años |
| Pct65ms | Población 65 años o más |

---

## Archivos principales del directorio `data`

| Archivo | Descripción |
|---|---|
| dataset.csv | dataset principal |
| dataset_with_indicators.csv | dataset con indicadores derivados |
| latest_snapshot.csv | última instantánea disponible |
| indicators_summary_by_country.csv | indicadores agregados por país |
| indicators_summary_by_year.csv | indicadores agregados por año |

## Posibles aplicaciones

- análisis de envejecimiento poblacional  
- estudio del bono demográfico  
- comparaciones regionales  
- generación de pirámides poblacionales  
- análisis de transición demográfica
