# Data - Latin America Population Age Structure Dataset

Main file: dataset.csv - 228 records (19 countries x 4 years x 3 sex categories), 41 variables, CSV/UTF-8.

Countries (19): Argentina, Bolivia, Brasil, Chile, Colombia, Costa Rica, Cuba, Ecuador, El Salvador, Guatemala, Honduras, Mexico, Nicaragua, Panama, Paraguay, Peru, Republica Dominicana, Uruguay, Venezuela.

Period: 2000-2015 (quinquennial: 2000, 2005, 2010, 2015). Unit of observation: Country x Year x Sex.

Additional files: dataset_with_indicators.csv, indicators_summary_by_country.csv, indicators_summary_by_year.csv.
