# Contribuciones al Dataset

Gracias por tu interés en contribuir a este conjunto de datos demográficos.

## Tipos de contribuciones aceptadas

Se aceptan contribuciones que mejoren la calidad científica del dataset, tales como:

- correcciones de errores en los datos
- incorporación de nuevas fuentes demográficas
- ampliación temporal del dataset
- mejora de la documentación
- generación de indicadores derivados
- visualizaciones o análisis reproducibles

## Requisitos para contribuciones

Toda contribución debe cumplir los siguientes criterios:

1. Las fuentes de datos deben ser oficiales o académicas.
2. Los cambios deben documentarse claramente.
3. Debe mantenerse la estructura original del dataset.
4. Las nuevas variables deben incluir documentación en el diccionario de datos.

## Formato de los datos

El dataset principal se mantiene en formato:

CSV

Las columnas deben conservar los nombres originales definidos en el esquema del dataset.

## Control de calidad

Antes de enviar una contribución se recomienda verificar:

- consistencia de porcentajes por grupos de edad
- coherencia de la población total
- ausencia de valores faltantes no justificados

## Cómo contribuir

1. Crear un fork del repositorio.
2. Realizar las modificaciones necesarias.
3. Documentar los cambios realizados.
4. Enviar un pull request.

## Citación del dataset

Si utilizas este conjunto de datos en una publicación académica, por favor cita el dataset utilizando el DOI correspondiente.
