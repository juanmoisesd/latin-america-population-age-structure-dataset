# Developer Guide

## Tech Stack
- **Language**: Python 3.10+
- **Data Manipulation**: Pandas, NumPy
- **Machine Learning**: Scikit-Learn (Linear Regression for forecasts)
- **Visualization**: Matplotlib, Seaborn
- **Templating**: Jinja2
- **Portal**: Static HTML/CSS/JS (Leaflet.js)

## Project Structure
- `data/`: Raw and processed CSV files.
- `scripts/`: Python logic for processing, stats, and site generation.
- `docs/`: Final build output (API, pages, assets).
- `main.py`: Orchestrator script.

## How to build
Run `python3 main.py` or `make all`.
